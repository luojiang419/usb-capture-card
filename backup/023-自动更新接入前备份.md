# 023-自动更新接入前备份

## 本阶段目标

- 参考 `D:\Myproject\vnt2.0\VntcApp1.0` 的 Android 更新链路，为 `usb摄像头` 接入应用内自动更新能力。
- 实现启动后检查 GitHub 最新 Release、下载 APK、提示安装更新。

## 修改前现状

### 已完成的发布链路

- GitHub Actions 已支持：
  - `push main` 自动构建 Artifact
  - `push v*` 自动发布 GitHub Release APK

### 应用端当前缺口

- 应用内还没有检查更新逻辑。
- `AndroidManifest.xml` 还没有声明：
  - `android.permission.INTERNET`
  - `android.permission.REQUEST_INSTALL_PACKAGES`
- `file_paths.xml` 目前只开放了录制视频目录：

```xml
<external-files-path
    name="usb_capture_movies"
    path="Movies/UsbCapture/" />
```

- `MainActivity.kt` 当前只有采集、录制、主题、分辨率与帧率设置逻辑，没有 GitHub Release 更新模块。
