# 003-USB预览前备份

## 阶段
模块三：USB 自动连接与预览开始前。

## 当前状态
- 模块二已完成，Compose UI、暗黑/浅色/跟随系统主题和基础动效可编译。
- AUSBC 已接入 `CameraClient + CameraUvcStrategy`，但运行时权限、设备详情和分辨率 UI 尚未完善。

## 本阶段计划修改
- 添加 Android 运行时 `CAMERA`、`RECORD_AUDIO` 权限请求。
- 授权后主动重新打开相机预览。
- 保存 `CameraClient` 与 `CameraUvcStrategy` 引用，用于读取设备、预览尺寸和切换分辨率。
- 设置 USB 设备连接回调，更新插入、授权、断开等状态。
- 在设置面板展示可用分辨率并支持切换。
