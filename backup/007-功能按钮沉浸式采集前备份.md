# 007-功能按钮沉浸式采集前备份

## 备份时间
- 2026-06-29

## 本阶段目标
- 将录制功能从常驻底部控制区收纳进功能按钮。
- USB 采集预览开始后默认隐藏控制层，保留全屏沉浸式采集画面。
- 返回键或滑动返回手势用于唤出功能按钮。
- 功能按钮菜单提供：录制/停止录制、采集设置、退出采集画面。
- 采集设置内提供采集卡返回的分辨率列表，以及 24/30/60fps 帧速率预设。

## 当前修改到哪个模块
- 模块七：功能按钮与沉浸式采集。

## 预计修改后的结构
- 新增 `showControls` 状态：预览成功时隐藏；返回键/手势显示；再次返回隐藏。
- 新增沉浸式系统栏方法：进入预览和隐藏控件时隐藏系统栏。
- 替换常驻顶部/底部控件为 `FunctionOverlay`：
  - 等待/错误/连接状态显示状态提示和“等待 USB 采集卡”按钮。
  - 预览状态下仅在 `showControls == true` 时显示“功能”按钮。
  - 按钮展开菜单：录制/停止录制、采集设置、退出采集画面。
- `SettingsPanel` 聚焦采集设置：设备、预览、状态、分辨率、帧速率。
