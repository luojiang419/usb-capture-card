# 012-等待状态UI精简前备份

## 备份时间
2026-06-29 23:28 后开始本阶段修改前。

## 当前阶段
模块十二：等待状态 UI 精简。

## 修改目标
- 未连接/等待采集卡时，只在屏幕中心显示 USB 图标和“等待采集卡”。
- 左上角不再额外显示独立的“等待采集卡”状态胶囊。
- 左上角功能按钮保留，并增加状态指示灯。
- 等待状态下不再显示误导性的“采集卡已连接 1280 × 720”副标题。

## 修改前关键代码

### FunctionOverlay：状态胶囊和功能按钮同时显示
```kotlin
Column(
    horizontalAlignment = Alignment.Start,
    verticalArrangement = Arrangement.spacedBy(10.dp)

}
```
