# 006-自动横竖屏适配

## 已完成内容
- 增加屏幕方向状态 `ScreenOrientation`，支持识别横屏和竖屏。
- 覆写 `onConfigurationChanged()`，横竖屏切换时同步 Compose UI 状态。
- 横竖屏切换后调用 `refreshRenderSize()`，把新的预览 View 宽高同步给 AUSBC 渲染层。
- Compose 控制层拆分为 `PortraitControls()` 和 `LandscapeControls()`：
  - 竖屏：保持原来的顶部状态、中间信号、底部录制控制。
  - 横屏：左侧显示状态和信号，右侧放置录制控制，更适合宽屏预览。
- 设置面板新增“方向”字段，方便真机验证当前是横屏还是竖屏。

### 新增：横竖屏变化回调
```kotlin
override fun onConfigurationChanged(newConfig: Configuration) {
    super.onConfigurationChanged(newConfig)
    screenOrientation = ScreenOrientation.from(newConfig.orientation)
    refreshRenderSize()
}
```

### 新增：刷新 AUSBC 渲染尺寸
```kotlin
private fun refreshRenderSize() {
    if (!::cameraView.isInitialized) return
    cameraView.post {
        if (cameraView.width > 0 && cameraView.height > 0) {
            cameraClient?.setRenderSize(cameraView.width, cameraView.height)
        }
    }
}
```

### 新增：横屏布局
```kotlin
if (screenOrientation == ScreenOrientation.Landscape) {
    LandscapeControls(...)
} else {
    PortraitControls(...)
}
```
