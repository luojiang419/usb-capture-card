# 012-左上功能按钮与锁屏防误触

## 已完成内容
- 将功能按钮从右下角移动到左上角。
- 全屏预览 USB 采集画面时，点击画面会在左上角半透明显示功能按钮。
- 功能按钮显示后，如果 3 秒无操作，会自动隐藏并回到沉浸式预览。
- 打开功能菜单或采集设置时不会自动隐藏，避免操作中断。
- 在右侧屏幕边缘中部新增锁定按钮：
  - 未锁定时显示解锁图标，点击后进入锁定。
  - 锁定时显示锁定图标，点击后解除锁定。
  - 锁定状态下点击画面不会唤出功能按钮，避免误操作。
- 采集断开、错误、停止预览时会自动解除锁定，避免等待页面被锁住。
- 按发布规则将版本从 `1.2.0` 升级到 `1.3.0`，`versionCode` 从 `3` 升到 `4`。
- 已执行构建：`.\gradlew.bat :app:assembleDebug`，构建成功。
- 最新 APK：`D:\Myproject\usb摄像头\app\build\outputs\apk\debug\USB采集卡.apk`
- 最新 APK 大小：`26190882` bytes。
- 构建元数据确认：
  - `versionCode`: `4`
  - `versionName`: `1.3.0`
  - `outputFile`: `USB采集卡.apk`
- 项目根目录 `.gradle` 临时缓存已清理；保留 `app/build`，因为其中包含最新 APK。

    uiState.isPreviewing,
    controlsVisible,
    functionMenuExpanded,
    showSettings,
    screenLocked,
    controlsInteractionTick
) {
    if (uiState.isPreviewing && controlsVisible && !functionMenuExpanded && !showSettings && !screenLocked) {
        delay(CONTROLS_AUTO_HIDE_MS)
        onAutoHideControls()
    }
}
