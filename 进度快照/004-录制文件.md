## 具体修改的代码前后对比

### 修改前：录制按钮占位
```kotlin
private fun handleRecordClick() {
    uiState = uiState.copy(errorMessage = "录制模块正在接入")
}
```

### 修改后：开始/停止真实录制
```kotlin
private fun handleRecordClick() {
    if (!uiState.isPreviewing) {
        uiState = uiState.copy(errorMessage = "未检测到视频信号")
        return
    }
    if (uiState.isRecording) {
        stopRecording()
    } else {
        startRecording()
    }
}
```

### 新增：录制保存
```kotlin
client.captureVideoStart(callback, target.absolutePath)
cameraClient?.captureVideoStop()
```

### 新增：FileProvider
```xml
<provider
    android:name="androidx.core.content.FileProvider"
    android:authorities="${applicationId}.fileprovider"
    android:exported="false"
    android:grantUriPermissions="true" />
```

### 新增：录制文件列表
```kotlin
RecordingFile(
    name = it.name,
    path = it.absolutePath,
    sizeBytes = it.length(),
    createdAtMillis = it.lastModified()
)
```
